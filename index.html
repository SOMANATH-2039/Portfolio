Machine Learning
Machine Learning

This course, I gained a comprehensive understanding of machine learning, data mining, and statistical pattern recognition. I explored supervised learning techniques, including parametric/non-parametric algorithms, support vector machines, kernels, and neural networks. I also studied unsupervised learning methods like clustering, dimensionality reduction, recommender systems, and deep learning. The course emphasized best practices in machine learning, such as bias-variance tradeoff and the innovation process in AI. Through case studies and applications, I learned to apply algorithms to real-world problems, including building smart robots, text understanding (e.g., web search, anti-spam), computer vision, medical informatics, and database mining.

NLP
Natural Language Processing

In this Specialization, I developed advanced natural language processing (NLP) skills through hands-on projects. I performed sentiment analysis of tweets using logistic regression and naïve Bayes, explored vector space models to uncover word relationships, and applied PCA to reduce dimensionality for visualization. I also created an English-to-French translation algorithm using word embeddings and locality-sensitive hashing for approximate k-nearest neighbor search. By the end, I had built NLP applications for question-answering, sentiment analysis, translation, and text summarization. The program was taught by experts Younes Bensouda Mourri from Stanford and Łukasz Kaiser from Google Brain, making it an enriching experience.
